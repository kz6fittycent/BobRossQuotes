# BobRossQuotes


A collection of quotes from Bob Ross. 

## To install:

`sudo snap install bobrossquotes`

Or use `pip`:
`pip install bobross==1.1`

## To run:

From your terminal: 
Snap command: `bobrossquotes`
If installed via `pip`: `bobross`
